# Game Design References

This folder contains selected and reorganized source material from the imported `gamedesign/` package plus AWKit-specific guidance for gamified apps.

Start with:
- `gamification-learning-loop.md` for Duolingo-like products.
- `game-development-planning.md` for pillars and scope.
- `game-design-methodology.md` for core loops and iteration.
- `reinforcement-feedback-systems.md` for rewards and punishment.
- `flow-state-design-framework.md` for difficulty curves.

The imported package remains useful as reference material, but the canonical runtime entry point is `../SKILL.md`.
